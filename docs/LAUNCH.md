# Launch kit

## One-line description

WhatsApp for Codex is a privacy-first local plugin that lets Codex message only you—or numbers you explicitly whitelist—and accept prefixed commands only from your WhatsApp self-chat.

## Short announcement

I built **WhatsApp for Codex**, an open-source local plugin with QR login, self-only sending by default, explicit recipient whitelisting, and an optional `/codex ` command inbox from your own WhatsApp self-chat. It exposes no contact-list, group, other-chat, or message-history tools. The goal is useful desktop automation without turning WhatsApp into an unrestricted messaging surface.

Repository: https://github.com/vkctata/codex-whatsapp

## Demo scenario

1. Send yourself: `/codex Find the cheapest flights from New York to London on 18 September.`
2. A recurring Codex task claims the owner-only command.
3. Codex uses the browser/search/travel tools installed on that desktop.
4. The plugin sends a concise comparison and source links back only to the owner.
